# 准备

## 图像坐标

对于一张宽高为 W x H 的图像来说，左上角 (0, 0)，右下角 (W, H)

举例：`*.tga`, `*.jpg`, ...


## 纹理坐标

左下角 (0, 0)，右上角 (1, 1)

举例：OpenGL, Metal, ...


## 屏幕坐标

中心 (0, 0)，左下角 (-1, -1), 右上角 (1, 1)

举例：OpenGL, Metal, ...
