## Round-off error

舍入误差，指计算机将信息量化存储时，因为用有限位数来表示实际上无限位数的信息，产生的舍入误差，属于量化误差。比如浮点数的精度是有限的，多步计算可能会出现舍入误差的积累，从而使得最后的结果存在问题。
